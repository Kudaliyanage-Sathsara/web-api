# web_API

This is a  CodeIgniter-based web API project. It runs under XAMPP (Apache + PHP + MySQL).

Quick start (local, Windows + XAMPP):

1. Put this folder in your XAMPP `htdocs` (already at `C:\xampp\htdocs\web_api`).
2. Make sure Apache and MySQL are running from the XAMPP control panel.
3. Create the database and tables from `database_setup.sql` (e.g., using phpMyAdmin).
4. Configure `application/config/database.php` as needed (DB credentials).
5. Visit `http://localhost/web_api/` in your browser.

To upload this project to GitHub, initialize a git repo, commit, and push (instructions in repository root or below).
